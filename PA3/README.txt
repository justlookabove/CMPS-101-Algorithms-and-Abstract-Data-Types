List.java - From PA1, contains abstract data type implemented as a doubly 
	    linked list that contains the Node class, constructors, access 
	    functions, manipulation functions, and other functions that 
	    defines the List object.

ListTest.java -  To test the List.java functions

Matrix.java - An abstract data type matrix, implemented as an array of Lists. 

MatrixTest.java - To test the Matrix.java functions

Sparse.java - Client Module, takes in values from an input file and prints 
	      out the manipulated matrices from Matrix.java.

Makefile - Creates an executable jar file called Sparse.  This makefile will 
	   compile all .java files in your current working directory.
